# N8 Group Homebrew Tap

Homebrew formulae for N8 Group tools.

## Install

```bash
brew tap n8group-oss/tap
brew install gh-audit
```

## Available Formulae

- **gh-audit** - GitHub audit, governance, and inventory for organizations
